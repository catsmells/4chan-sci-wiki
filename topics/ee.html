Electronics and electrical engineering

   

starting electronics

To start learning electronics and electrical theory you will need to start with basic electronics.
Grob's by Mitchel E. Schultz is a good start. The book covers everything an electrical technician needs. To start with electronics you have to understand volts, ohms,amps and power You should be able to find voltage drop and total current. you must know why these values matter.

Tools of the trade

If you want to tinker with electronics and see the theory in action some tools will be needed.
if you are just getting started here are some extreme basics:

  • breadboard
  • power supply
  • jumper cables(male to male) (don't get female ones)
  • Resistors
  • multimeter (for hobby or starter projects 20USD+ is fine. as you do more projects consider upgrading to a fluke)

Tools for Digital electronics

if you want to get deeper into Digital electronics i would recommend these components along with the above

  • 74 series ICs(make sure they are DIP
  • 4000 series ICs (DIP too)
  • 555 timers
  • datasheets for your components

Soldering

  • adjustable soldering iron
  • solder
  • desoldering braids(i personally never had luck with the desoldering
  • flux
  • Gloves(heat resistant)
  • helping hands or gripper

As you start with projects you shouldn't need an oscilloscope or function generator.
In do it yourself or personal use situations practical wisdom states you buy AS NEEDED!!

This is not a comprehensive guide. Never work on live equipment unless you are an electrician. If you need any help /diy/sqt,/diy/ohm and the sci #electronics discord are excellent places to ask
